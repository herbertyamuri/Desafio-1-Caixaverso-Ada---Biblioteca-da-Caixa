CAIXA | Biblioteca

Bem-vindo(a) ao Sistema de Gestão da Biblioteca da Caixa.

Para interagir com o sistema, abra o console do desenvolvedor (pressione F12).

Diagrama UML Simplificado do Projeto

Você pode usar o objeto biblioteca que foi exposto globalmente para chamar as funções conforme a lista abaixo:

Lista de Funções

biblioteca.listarLivros()

Lista todos os livros cadastrados no sistema, mostrando título, autor, disponibilidade e ISBN.

biblioteca.listarAutores()

Lista todos os autores cadastrados com seus respectivos dados.

biblioteca.listarUsuarios()

Lista todos os usuários cadastrados no sistema.

Cadastro de Autores

biblioteca.cadastrarAutor("Nome do Autor", "Nacionalidade", anoNascimento)

Cadastra um novo autor no sistema.

  • Nome do Autor: texto entre aspas
  • Nacionalidade: texto entre aspas
  • anoNascimento: número do ano (ex: 1964)

Cadastro de Livros

biblioteca.cadastrarLivro("Título do Livro", "Nome do Autor", anoPublicacao, "Gênero", "ISBN")

Cadastra um novo livro no sistema.

  • Título do Livro: texto entre aspas
  • Nome do Autor: deve ser um autor já cadastrado
  • anoPublicacao: número do ano
  • Gênero: texto entre aspas
  • ISBN: código ISBN entre aspas

Gerenciamento de Empréstimos

biblioteca.emprestarLivro("ISBN-do-livro", "matricula-do-usuario")

Registra o empréstimo de um livro para um usuário.

  • ISBN-do-livro: código ISBN do livro entre aspas
  • matricula-do-usuario: número de matrícula entre aspas
biblioteca.devolverLivro("ISBN-do-livro")

Registra a devolução de um livro.

  • ISBN-do-livro: código ISBN do livro entre aspas

Administração (Apenas para Usuários Admin)

biblioteca.excluirLivro("ISBN-do-livro", adminExecutor)

Remove um livro do sistema (requer objeto adminExecutor).

  • ISBN-do-livro: código ISBN do livro entre aspas
  • adminExecutor: objeto do tipo Admin
